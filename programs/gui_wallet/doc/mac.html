Steps for compiling 'gui_wallet' application on MAC OS



1. Check existance of QT5. If it does not exist, then install it (qt4 also should work)

2. Xcode set up properly (follow the link or accoprding to Vazgen this should also work 'sudo xcode-select -switch /Applications/Xcode.app/Contents/Developer')

3. Set up following environmental variables (export VAR_NAME=VAR_VALUE)
        a) DECENT_ROOT (Default: '../../../../DECENT-Network')         # the path to the root of DECENT-Network
        b) BOOST_ROOT (Default: '/usr/local/opt/boost')                       # path, where boost 'lib' and 'include' folders are located
        c) OPENSSL_ROOT_DIR (Default: '/usr/local/opt/openssl')       # path for openssl stuff
        d) CRIPTOPP_ROOT_DIR (Default: '/usr/local/opt/cryptopp')   # path for openssl stuff
4. before running the app set up DYLD_LIBRARY_PATH environmental variable (export DYLD_LIBRARY_PATH=$DYLD_LIBRARY_PATH:$BOOST_ROOT/lib)
